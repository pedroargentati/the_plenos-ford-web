Big Query

  • Manutenção: Atualização estatística automática

Escolhemos o BigQuery pelo fato da Ford já utilizar este Banco de Dados e poderemos reutilizar a instância deles. Além deste fato, o BigQuery é um dos melhores do mercados, e um dos mais atuais, assim gerando confiabilidade e qualidade.

Arquitetura Ford Fuel

A seguir, segue as explicações das tecnologias usadas para o desenvolvimento, com suas específicações e aplicabilidade no projeto.

Java Spring Boot

Será utilizado o SDK 17 do Java no desenvolvimento de APIs (Application Programming Interfaces), seguindo a arquitetura de microsserviços.

API de Scraping em Python

Uma requisição GET será originada a partir do back-end da aplicação, contendo o endereço como parâmetro. Esse endereço será empregado nas buscas realizadas pelo programa de web scraping programado. Se o nome do estabelecimento for encontrado, ele será devolvido à aplicação de origem.

Front-End em Flutter

Será utilizado no Visual Studio Code para criar as telas adicionais e fazer modificações em algumas telas do aplicativo Ford Pass, a fim de incluir funcionalidades extras.

Power BI

Será utilizado para integrar a base de dados e automatizar a criação de relatórios semanais através de dashboards.

Transação de dados para o Banco de Dados

Para enviar esses dados para o banco de dados, usaremos REST APIs, que são requisições que permitem a comunicação de dados entre aplicações.

Por que utilizar Ford Fuel ?

Nossa solução propõe a integração do medidor de qualidade de combustível diretamente no sistema dos carros. Isso significa que os motoristas terão acesso imediato às informações sobre a qualidade do combustível no painel do carro, tornando o processo mais conveniente e rápido

Análise de Dados e Geração de Relatórios

Coletamos e armazenamos dados sobre a qualidade do combustível ao longo do tempo, permitindo a geração de relatórios detalhados e a análise de tendências. Essas informações podem ser utilizadas para identificar padrões, otimizar o desempenho dos veículos e tomar decisões informadas sobre a escolha de postos de combustível.

Integração com o Aplicativo FordPass

O FordPass é um aplicativo inovador que oferece uma variedade de recursos para os motoristas, com a integração da nossa solução, os usuários poderão acessar facilmente informações cruciais sobre a qualidade do combustível diretamente em seus smartphones.
